# Curso de Tratamientos Faciales para Ivania

Curso de refuerzo de 12 semanas para estudiar 1 hora diaria de lunes a jueves.

## Como usar este curso

1. Abrir `00 - Inicio.md`.
2. Estudiar una leccion por dia.
3. Usar la bitacora con cada modelo real.
4. Repasar flashcards antes de cada sesion.
5. Importar los CSV de `Notion/` cuando se quiera llevar el curso a Notion.

## Enfoque

- Reforzar Tratamientos Faciales de AMEVIE.
- Mejorar diagnostico, ingredientes y teoria.
- Practicar con casos y modelos reales.
- Usar Videlca, Ainhoa y Miguett como material didactico.
- Mantener seguridad: no sustituye clases, supervision, manuales de equipo ni atencion medica.

## Estructura

- `Lecciones/`: 48 sesiones listas para estudiar.
- `Material de apoyo/`: glosario, seguridad, ingredientes, protocolos, casos y recursos.
- `Notion/`: tablas CSV compatibles con bases de datos de Notion.
- `Visuales/`: 48 laminas SVG, una por leccion, para estudiar de forma visual.
- `App/`: pagina local interactiva con progreso, quizzes, flashcards, casos con imagenes y recursos.

## Cambios v2

Esta versión corrige rutas con acentos, agrega una vista de Diagnóstico guiado y muestra un semáforo de seguridad en lecciones, casos y protocolos.

Para abrir la app local:

1. Abre una terminal en la carpeta `Tratamientos Faciales - Ivania`.
2. Ejecuta un servidor local, por ejemplo:
   - `python -m http.server 8787` si tienes Python funcionando.
   - O cualquier servidor local equivalente.
3. Abre `http://127.0.0.1:8787/App/`.

No abras `index.html` directo con doble clic, porque el navegador puede bloquear la carga de CSVs por seguridad.
