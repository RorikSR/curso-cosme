# Ingredientes cosmeticos

## Como estudiar ingredientes

Leer cada producto por funcion, no por promesa comercial.

1. ¿Que problema estetico intenta apoyar?
2. ¿Que activo principal contiene?
3. ¿Para que tipo de piel parece adecuado?
4. ¿En que piel lo evitaria o usaria con cuidado?
5. ¿En que paso del protocolo entra?

## Marcas

- Videlca: usar para estudiar activos como acido hialuronico, vitamina C, niacinamida, AHA y productos profesionales.
- Ainhoa: usar para hidratacion, sensorialidad, acido hialuronico, piel sensible y antiedad suave.
- Miguett: usar para limpieza, hidratacion, sueros, niacinamida, vitamina C y fotoproteccion.
